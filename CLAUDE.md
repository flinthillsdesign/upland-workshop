# Upland Workshop

Central repository for shared notes, patterns, and cross-cutting work across the Upland Workshop suite of applications.

## Applications

- **Upland Inquiries** — Customer inquiry management
- **OAuth** — Centralized authentication/authorization service
- **Previews** — Content preview system
- **Scheduler** — Scheduling and calendar management
- **Upland Storefront** — Customer-facing website, backend services, and order management

## Repository Purpose

This repo serves as:
- A shared knowledge base for patterns and conventions used across all apps
- A place to document architectural decisions that span multiple projects
- A workspace for prototyping and developing shared components
- Documentation of the philosophies and principles guiding development
